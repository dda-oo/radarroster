# RadarRoster - AI & Data Solutions

A sleek, modern, one-page consulting website showcasing **AI, Data Engineering, and Cloud Modernization** services.

**Live Site:** [radarroster.com](https://radarroster.com)

---

## Features

- **Responsive Design** – Optimized for all devices  
- **Modern Tech Stack** – HTML5, Tailwind CSS, Vanilla JavaScript  
- **Smooth Animations** – Scroll-based and hover effects  
- **Functional Contact Form** – Integrated with [Web3Forms](https://web3forms.com/) and hCaptcha  
- **SEO Optimized** – Semantic HTML structure  
- **Fast Loading** – Minimal dependencies, CDN-based  
- **GitHub Pages**  

---

## Tech Stack

- HTML5  
- Tailwind CSS  
- Vanilla JavaScript  
- No build process required

---

## Contact Form

The contact form includes:  

- Honeypot spam protection  
- Client-side validation  
- Success/error feedback  
- Secure access key handling
